# Birthday Website Generator - Usage Guide

This tool lets you create a customized birthday countdown website with music, photos, and messages.


## Live Demo

[Click here to view the live demo](https://gaurav-kamti.github.io/Birthday_studio/)

## How to Use

1. **Open the Editor**
   - Open `editor.html` in your web browser.

2. **Fill in Details**
   - Enter the Birthday Person's Name.
   - Set the Birthday Date & Time.
   - Enter Your Name.

3. **Upload Music**
   - **Before Birthday**: Plays during the countdown.
   - **On Time**: Plays when the birthday arrives.
   - **After**: Plays on the gift page.

4. **Upload Photos**
   - Select multiple photos for the memory gallery slideshow.

5. **Upload Memes/GIFs**
   - Select fun images for Stage 1 of the gift page.

6. **Write Messages**
   - Add a special poem.
   - Write a personal letter.

7. **Add Friend Messages**
   - Click "+ Add Friend Message".
   - Enter friend's name and their wish.

8. **Get Your Link**
   - Click "Generate Birthday Website".
   - A shareable link will be generated (e.g., `.../index.html?data=...`).
   - Copy this link.

## Important Note

**Images & Music Limitation**:
Because this tool creates a direct link without a server, **uploaded photos and music cannot be saved in the link**.
- The link will only save your **Names, Dates, Messages, Poem, and Theme**.
- If you want to include custom photos, you must use the manual download & upload method.

## Sharing

Simply send the generated link to the birthday person!
- The generated website allows you to bookmark it or share the URL immediately.
- Ensure you upload photos and memes for the best experience.

## Sharing

Once generated:
- Bookmark the page.
- Share the URL with the birthday person.
- Keep the window open to access it anytime.
